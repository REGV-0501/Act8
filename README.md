## Prompt:
IDE: antigravity 
Framework: Flutter
Lenguaje de programación: Dart
Proyecto Datos de Producto

Descripción de proyecto:

Crear un formulario para capturar 6 datos de un producto y un boton para guardar dichos datos, una clase producto con atributos(id, nombre, precio, categoria, descripcion, stock disponible), que inicializar un diccionario vacío llamado datosproducto, que el id sea auto numérico, conforme se vayan registrando se guarden en el diccionario, crear los archivos, main.dart, claseproducto.dart, diccionarioproducto.dart, verproducto.dart,crear un agente que se llame guardardatosproducto.dart para modificar el archivo diccionarioproducto.dart nivel principiante, formulario atractivo y elegante, buena distribucion de los widgetes, utilizar rutas nombradas para navegar entre pantallas, inicio.dart en ella con dos botones uno para capturar productos  capturaproductos.dart, y otro botón para ver productos verproductos.dart , que se totalmente funcional. Utiliza una gama de colores guinda para el fondo, botones de color verde, colores claros para las letras.
